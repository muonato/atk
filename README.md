# Automation Toolkit
Simplified script deployment for Windows Powershell
- Pull your hostname-specific scripts to server with ease
- Organize your script repository to server groups
- Server maintenance task cataloging made easy

# Quickstart
1. Add your Powershell script to 'script\'
2. Add script filename to 'config\tasks.ini'
3. Add server hostname to 'config\hosts.ini'
4. Start 'app.ps1' script from target host
